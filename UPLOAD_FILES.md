# GitHub에서 파일 업로드하기

## 현재 위치
파일 편집 페이지에 있습니다. 파일 업로드 페이지로 이동해야 합니다.

## 방법 1: 저장소 메인 페이지로 이동

1. 브라우저 주소창에서 다음 URL로 이동:
```
https://github.com/realks2002-cell/bridge-test
```

2. 저장소 메인 페이지에서:
   - 상단에 **"Add file"** 버튼이 보입니다
   - 클릭 → **"Upload files"** 선택

## 방법 2: 직접 업로드 URL 사용 (가장 빠름!)

브라우저 주소창에 다음 URL을 입력하세요:

```
https://github.com/realks2002-cell/bridge-test/upload/main
```

이 URL로 바로 파일 업로드 페이지로 이동합니다!

## 방법 3: 현재 페이지에서

1. 브라우저의 **뒤로 가기** 버튼 클릭
2. 또는 주소창에서 `/new/main` 부분을 `/upload/main`으로 변경

## 파일 업로드 페이지에서

1. 파일을 드래그 앤 드롭하거나
2. "choose your files" 클릭하여 파일 선택
3. 하단 "Commit changes" 클릭

## 업로드할 파일들

다음 폴더/파일들을 업로드하세요:

```
app/              (전체 폴더)
components/       (전체 폴더)
lib/              (전체 폴더)
types/            (전체 폴더)
public/           (전체 폴더, CSV 파일 포함)
package.json
next.config.ts
tsconfig.json
tailwind.config.js
postcss.config.mjs
vercel.json
README.md
.gitignore
```

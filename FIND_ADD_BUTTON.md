# GitHub에서 "Add file" 버튼 찾기

## 방법 1: 빈 저장소에서 파일 추가

### 현재 화면에서:
1. 페이지 중앙의 **"Get started by creating a new file or uploading an existing file"** 텍스트를 찾으세요
2. 이 텍스트를 클릭하거나
3. 또는 페이지에서 **"creating a new file"** 링크를 클릭

### 또는:
1. 저장소 페이지에서 **"Code"** 탭이 선택되어 있는지 확인
2. 빈 저장소에서는 중앙에 큰 텍스트 박스가 보입니다
3. 그 안의 **"creating a new file"** 또는 **"uploading an existing file"** 링크 클릭

## 방법 2: 직접 URL 접속

저장소 페이지에서 다음 URL로 직접 이동:
```
https://github.com/realks2002-cell/bridge-test/upload
```

또는:
```
https://github.com/realks2002-cell/bridge-test/new/main
```

## 방법 3: README 파일 먼저 생성

1. 저장소 페이지에서 **"creating a new file"** 클릭
2. 파일명: `README.md` 입력
3. 내용 입력 (선택사항)
4. "Commit new file" 클릭
5. 이제 상단에 **"Add file"** 버튼이 보입니다!

## 방법 4: 파일이 있는 저장소에서

파일이 하나라도 있으면:
- 저장소 페이지 상단에 **"Add file"** 버튼이 보입니다
- "Add file" → "Upload files" 선택

## 빠른 방법: 직접 업로드 URL

브라우저 주소창에 입력:
```
https://github.com/realks2002-cell/bridge-test/upload/main
```

이 URL로 바로 파일 업로드 페이지로 이동합니다!

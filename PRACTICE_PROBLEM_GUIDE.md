**장점**: 가장 실용적이고 즉시 활용 가능

**예시 문제**:

## CSV 구조 변경 제안

### 현재 구조
```
문제번호,문제,선택지1,선택지2,선택지3,선택지4,정답,해설
```

### 실습 문제용 구조 (제안)
```
문제번호,문제유형,문제,데이터셋URL,코드템플릿,정답코드,해설,난이도
```

**필드 설명**:
- `문제유형`: "데이터분석", "전처리", "시각화", "통계", "머신러닝"
- `데이터셋URL`: 실습용 데이터셋 URL
- `코드템플릿`: 학생이 작성할 코드의 기본 틀
- `정답코드`: 정답 코드 (실행 결과 비교용)
- `난이도`: "초급", "중급", "고급"

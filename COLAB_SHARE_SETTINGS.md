# Colab 노트북 공유 설정 가이드

## 현재 문제

공유 설정이 "액세스 권한이 있는 사용자"로 되어 있어서:
- ❌ 다른 사람들이 접근할 수 없음
- ❌ QR코드로 접속한 독자들이 노트북을 열 수 없음

## 해결 방법

### 1단계: 공유 설정 변경

현재 화면에서:

1. **"일반 액세스" 섹션** 찾기
2. **"제한됨"** 또는 **"액세스 권한이 있는 사용자"** 옆의 **드롭다운** 클릭
3. **"링크가 있는 모든 사용자"** 선택

### 2단계: 권한 설정

1. **"뷰어"** 또는 **"편집자"** 선택
   - **뷰어**: 읽기만 가능 (권장)
   - **편집자**: 수정 가능
2. **"완료"** 또는 **"저장"** 클릭

### 3단계: 공유 링크 복사

1. 공유 설정 화면에서 **링크 복사** 버튼 클릭
2. 또는 노트북 URL에서 노트북 ID 확인:
   ```
   https://colab.research.google.com/drive/YOUR_NOTEBOOK_ID
   ```
   `YOUR_NOTEBOOK_ID` 부분을 복사

### 4단계: 웹사이트에 URL 추가

`lib/config.ts` 파일에서:

```typescript
{
  id: 1,
  title: '1회차 모의고사',
  colabUrl: 'https://colab.research.google.com/drive/복사한_노트북_ID',
  csvFilename: 'exam1.csv',
  description: 'AICE 자격증 1회차 모의고사',
},
```

## 설정 확인

공유 설정이 올바르게 되었는지 확인:

1. **시크릿 모드**로 브라우저 열기
2. Colab 노트북 URL 접속
3. 노트북이 열리면 → ✅ 성공!
4. 접근 거부 오류가 나면 → ❌ 공유 설정 다시 확인

## 각 모의고사별 설정

각 모의고사마다 별도의 Colab 노트북을 만들고:
1. 각 노트북을 "링크가 있는 모든 사용자"로 공유
2. 각 노트북의 ID를 `lib/config.ts`에 추가

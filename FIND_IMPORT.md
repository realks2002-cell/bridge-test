# Vercel에서 프로젝트 추가하기

## 현재 위치
Vercel 대시보드에 로그인되어 있습니다. "Import Project" 버튼을 찾아야 합니다.

## 프로젝트 추가 방법

### 방법 1: "Import Project" 버튼 찾기

화면에서 다음을 찾으세요:

1. **"Projects"** 섹션 찾기
2. **"Deploy your first project"** 또는 **"Import Project"** 버튼 클릭
3. 또는 화면 상단의 **"Add New..."** 버튼 클릭

### 방법 2: 직접 URL 접속

브라우저 주소창에 다음 URL을 입력:

```
https://vercel.com/new
```

이 URL로 바로 프로젝트 추가 페이지로 이동합니다!

### 방법 3: 상단 메뉴에서

1. Vercel 대시보드 상단 메뉴 확인
2. **"Add New..."** 또는 **"+"** 버튼 클릭
3. **"Project"** 선택

## 프로젝트 추가 후

1. **"Import Git Repository"** 선택
2. GitHub 저장소 목록에서 **`realks2002-cell/bridge-test`** 찾기
3. **"Import"** 클릭
4. Project Name: `bridge-mock-exam` 입력
5. **"Deploy"** 클릭

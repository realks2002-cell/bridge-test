# 프로젝트 목적 상세 설명

## 프로젝트 목적

**IT 수험서(AICE, 빅데이터 분석기사) 독자들이 책에 있는 QR코드를 통해 손쉽게 실습 환경(Google Colab)에 접속하여 모의고사를 풀 수 있도록 돕는 '브릿지(Bridge) 웹사이트' 구축**

## 핵심 개념

### 브릿지(Bridge) = 다리 역할
```
책의 QR코드 → 웹사이트 → Google Colab
              ↑
         이 부분이 브릿지!
```

웹사이트는 **중간 다리** 역할을 합니다:
- 독자가 QR코드로 쉽게 접속
- 인증 처리
- Colab으로 연결해주는 역할

## 전체 흐름

### 1단계: QR코드 스캔
- 독자가 책에 있는 QR코드를 스캔
- 모바일에서 접속

### 2단계: 웹사이트 접속
- 모바일 안내 페이지 표시
- PC 접속 안내 + URL 복사

### 3단계: PC에서 접속
- 인증 코드 입력 (도서 내 인증 코드)
- 모의고사 리스트 확인

### 4단계: Colab 접속
- CSV URL 복사
- **Colab 노트북 열기** 버튼 클릭
- Google Colab 환경으로 이동

### 5단계: Colab에서 문제 풀이
- **여기서 실제 문제를 풉니다!**

## Colab에서 무엇을 하나요?

### AICE / 빅데이터 분석기사 시험 구성

이 자격증들은 **2가지 유형**의 문제가 있습니다:

#### 1. 이론 문제 (객관식)
- 파이썬 문법, 라이브러리 사용법 등
- **웹에서 풀 수 있음** (현재 시스템)
- 또는 Colab에서도 풀 수 있음

#### 2. 실기 문제 (파이썬 코딩) ⭐
- **이게 Colab에서 하는 주요 일!**
- 실제 파이썬 코드를 작성하고 실행
- 데이터 분석, 머신러닝 등 실무 작업

### Colab에서 하는 구체적인 작업

#### 예시 1: 데이터 분석 문제
```python
# 문제: 주어진 데이터의 평균을 구하시오
import pandas as pd

# CSV 데이터 로드 (웹에서 복사한 URL 사용)
df = pd.read_csv('https://bridge-mock-exam-nextjs.vercel.app/data/exam1.csv')

# 평균 계산
result = df['정답'].mean()
print("평균:", result)
```

#### 예시 2: 데이터 시각화 문제
```python
# 문제: 데이터를 시각화하시오
import matplotlib.pyplot as plt

# 데이터 로드
df = pd.read_csv('...')

# 그래프 그리기
plt.plot(df['문제번호'], df['정답'])
plt.show()
```

#### 예시 3: 머신러닝 문제
```python
# 문제: 주어진 데이터로 모델을 학습시키시오
from sklearn.linear_model import LinearRegression

# 데이터 준비
X = df[['feature1', 'feature2']]
y = df['target']

# 모델 학습
model = LinearRegression()
model.fit(X, y)

# 예측
predictions = model.predict(X)
```

## 웹사이트 vs Colab 역할 분담

### 웹사이트 (브릿지 역할)
- ✅ 인증 처리
- ✅ 모의고사 리스트 제공
- ✅ CSV URL 제공
- ✅ Colab 링크 제공
- ✅ (선택사항) 이론 문제 풀이

### Colab (실습 환경)
- ✅ CSV 데이터 로드
- ✅ 파이썬 코드 작성
- ✅ 코드 실행 및 결과 확인
- ✅ 실기 문제 풀이
- ✅ 데이터 분석, 시각화 등

## 실제 사용 시나리오

### 시나리오 1: 이론 문제만 있는 경우
1. 웹에서 CSV URL 복사
2. Colab에서 데이터 로드
3. 문제를 읽고 답안 선택
4. 결과 확인

### 시나리오 2: 실기 문제가 있는 경우 (주요 목적)
1. 웹에서 CSV URL 복사
2. Colab 노트북 열기
3. **파이썬 코드 작성**
4. 코드 실행
5. 결과 확인 및 제출

## 왜 Colab을 사용하나요?

### 장점
1. **설치 불필요**: 브라우저만 있으면 됨
2. **무료**: Google 계정만 있으면 사용 가능
3. **파이썬 환경**: 데이터 분석에 필요한 모든 라이브러리 포함
4. **공유 용이**: 노트북을 공유하여 학생들이 쉽게 접근
5. **실습 환경**: 실제 코드를 작성하고 실행할 수 있음

## 현재 시스템의 문제점

현재 시스템은:
- 웹에서 직접 문제를 풀 수 있게 되어 있음
- Colab 버튼이 없음

하지만 원래 목적은:
- **Colab에서 문제를 풀도록** 하는 것

## 해결 방안

### 옵션 1: 원래 목적대로 (권장)
- 웹: 인증 + 리스트 + Colab 링크 제공
- Colab: 실제 문제 풀이
- 웹에서 직접 풀기는 선택사항

### 옵션 2: 하이브리드
- 웹: 이론 문제 (객관식)
- Colab: 실기 문제 (파이썬 코딩)

## 결론

**Colab에서 하는 일:**
1. CSV 데이터 로드
2. **파이썬 코드로 실기 문제 풀이** (핵심!)
3. 코드 실행 및 결과 확인
4. 데이터 분석, 시각화 등 실무 작업

**웹사이트는:**
- Colab으로 연결해주는 **다리** 역할
- 인증, 리스트 제공 등 **편의 기능** 제공

즉, **실제 문제 풀이는 Colab에서, 웹사이트는 접근을 돕는 역할**입니다!

## 문제 해결

### CSV 로드 오류
- CSV 파일이 올바른 형식인지 확인
- 컬럼명이 정확한지 확인 (문제유형, 문제, 데이터셋URL, 코드템플릿, 정답코드, 해설, 난이도)

### 코드 실행 오류
- 데이터셋 URL이 접근 가능한지 확인
- 코드 템플릿에 '데이터셋URL' 문자열이 포함되어 있으면 실제 URL로 교체 필요

### 정답 비교 오류
- 숫자 비교는 부동소수점 오차를 고려 (0.0001 이내)
- 문자열 비교는 공백 제거 후 비교

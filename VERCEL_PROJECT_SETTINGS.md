# Vercel 프로젝트 설정 확인

## 현재 상태
프로젝트 `bridge-test`가 이미 배포되어 있습니다:
- URL: `bridge-test-nine.vercel.app`
- 저장소: `KS-Tech-DEPT/bridge-test`
- 배포 시간: 7분 전

## 프로젝트 설정 확인

### 1. 프로젝트 설정 페이지로 이동

1. **`bridge-test`** 프로젝트 클릭
2. 상단 메뉴에서 **"Settings"** 탭 클릭
3. 왼쪽 사이드바에서 **"General"** 선택

### 2. 프로젝트 이름 확인/변경

**Settings → General**에서:
- **Project Name**: 여기서 프로젝트 이름 확인 및 변경 가능
- 현재: `bridge-test`
- 변경 가능: `bridge-mock-exam` 등

### 3. Git 저장소 연결 확인

**Settings → Git**에서:
- 연결된 저장소 확인
- 필요 시 저장소 변경 가능

## 배포된 사이트 확인

배포된 URL로 접속:
```
https://bridge-test-nine.vercel.app
```

또는 프로젝트 페이지에서:
1. **"Deployments"** 탭 클릭
2. 최신 배포 클릭
3. **"Visit"** 버튼 클릭

## 새 프로젝트로 추가하고 싶다면

현재 프로젝트를 그대로 사용하거나, 새로 만들 수 있습니다:

1. **"Add New..."** 버튼 클릭
2. **"Import Git Repository"** 선택
3. 저장소 URL 입력: `https://github.com/realks2002-cell/bridge-test`
4. **"Private Repository Name"** 필드에: `bridge-mock-exam` 입력
5. **"Deploy"** 클릭

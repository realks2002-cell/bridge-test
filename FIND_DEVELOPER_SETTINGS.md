# Personal Access Token 생성하기

## 현재 위치
저장소 Settings 페이지에 있습니다. Personal Access Token은 **개인 계정 Settings**에서 생성해야 합니다.

## 올바른 경로

### 방법 1: 프로필에서 이동

1. GitHub 페이지 **우측 상단**의 **프로필 아이콘** 클릭
2. **"Settings"** 클릭 (개인 계정 설정)
3. 왼쪽 사이드바 **맨 아래**로 스크롤
4. **"Developer settings"** 클릭
5. **"Personal access tokens"** → **"Tokens (classic)"** 클릭
6. **"Generate new token (classic)"** 클릭

### 방법 2: 직접 URL 접속

브라우저 주소창에 다음 URL을 입력:

```
https://github.com/settings/tokens
```

또는:

```
https://github.com/settings/developers
```

## 단계별 스크린샷 설명

1. **프로필 아이콘**: GitHub 페이지 우측 상단 (원형 아이콘)
2. **Settings**: 프로필 메뉴에서 "Settings" 선택
3. **Developer settings**: Settings 페이지 왼쪽 사이드바 맨 아래
4. **Personal access tokens**: Developer settings 안에 있음

## 주의사항

- **저장소 Settings** (현재 위치) ❌
- **개인 계정 Settings** (여기로 이동 필요) ✅

## 빠른 링크

개인 계정 Settings로 이동:
```
https://github.com/settings/profile
```

Personal Access Tokens 페이지로 직접 이동:
```
https://github.com/settings/tokens
```
